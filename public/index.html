Image Search Abstraction Layer

User stories:

  1. I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
      Ex. https://mdasilva-imgsearch.herokuapp.com/api/imagesearch/<insert search term here>


  2. User can paginate through the responses by adding a ?offset=2 parameter to the URL.
      Ex. https://mdasilva-imgsearch.herokuapp.com/api/imagesearch/<insert search term here>?offset=<offset amount>
  3. User can get a list of the most recently submitted search strings.
      Ex. https://mdasilva-imgsearch.herokuapp.com/api/latest/imagesearch/

    4. Latest Searches!
